# Backend (Laravel)

Este diretório contém a base para o backend em Laravel. Para gerar o projeto execute:

```bash
composer create-project laravel/laravel .
```

Após a criação, implemente os controladores e modelos conforme os esboços gerados em `app/Http/Controllers` e `app/Models`. Todos os modelos do código CakePHP original possuem um controlador correspondente.
